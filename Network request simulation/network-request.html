Mini-Exercise: Simulating a Network Request

In this exercise, you'll create a Promise that simulates fetching data from a server. You'll use setTimeout to mimic the delay of a network request. Follow the instructions in the JavaScript comments to complete the task.

  • If the condition is true, resolve the Promise with a message "Data retrieved successfully!"
  • If the condition is false, reject the Promise with an error message "Data retrieval failed!"

After saving your changes, press the "Run" button to test your code. The result of your Promise will be displayed in the red box below. Make sure you see the correct message based on the condition you set!
